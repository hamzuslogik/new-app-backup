# Frontend React - CRM JWS Group

Application React moderne pour le système CRM.

## Installation

```bash
npm install
```

## Démarrage

```bash
npm run dev
```

L'application sera accessible sur `http://localhost:3000`

## Build pour production

```bash
npm run build
```

Les fichiers seront générés dans le dossier `dist/`

## Structure

- `src/components/` - Composants réutilisables
- `src/pages/` - Pages de l'application
- `src/contexts/` - Contextes React (Auth, etc.)
- `src/config/` - Configuration (API, etc.)

## Technologies

- React 18
- React Router
- React Query
- Vite
- Axios
